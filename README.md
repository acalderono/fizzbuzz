
# Test Fizz Buzz




## Run Locally

Clone the project

```bash
  git clone https://github.com/acalderono/fizzbuzz.git
```

Go to the project directory

```bash
  cd fizzbuzz
```

Start

```bash
  node fizzbuzz
```

Test
```bash
 npm install
 npm run test
```
